黑客与言论

受到影响而不自知，是非常危险的

  • 所谓时尚，本质上就是自己看不见自己的样子
  • 流行的不仅有衣服，还有道德观念
  • 不管哪一个年代，都是“祸从口出”
  • 任何一个年代的人，历史的常态都是，他们会对一些荒谬的东西深信不疑，只要有人稍微表示一点怀疑，就会惹来大麻烦

所谓大流，所谓”不能说的话“

  • 没有那么多巧合，你毫不犹豫的观点，深深赞同的观点，很多时候来自别人，你把别人灌输的观点当作了自己的观点
  • 古代地图工匠会故意画错一个小地方，抄袭者无处遁形
  • 独立思考的观点和社会主流一致？那么别人犯错了，你也必须犯同一个的错误
  • 如果你也犯下和别人一样的错误，那么这个错误不太可能完全来自于你自己
  • 如果别人告诉你什么，你就相信，那么你就会和别人犯一样的错误

第一种方法：真伪

  • 周围的人因为说了什么而陷入麻烦？
  • 触怒他人的言论，是那些可能会有人相信的言论

第二种方法：特别

  • “不能说的话”除了真话，还可能因为它非常特别。于是被贴上了“标签”
    1. “异端邪说”，贴上标签是想要在人们开始思考他们是否真实之前封杀他们
    2. “失败主义者”鲁登道夫、丘吉尔给政治对手贴上标签，指责他们奉行投降主义，以此借口将他们清除出政府。被贴上标签、受到打压后，人们都噤若寒蝉，这是很理想的压制反对声音的办法
    3. 万金油的“不适当”到和可怕的“制造分裂”
    4. 不谈论对错，而是直接贴标签，这就是你需要额外注意的地方
  • 最先从你头脑中跳出来的想法，往往就是最困扰你、很可能为真的想法，你注意到了，但是没有认真思考过
  • 头脑深处可能已经意识到哪些地方有问题，但是这种深层的反应可能没有上升成为自觉的意识

第三种方法：回顾过去

  • 过去那些被认为理所当然，如今却被认为不可思议的事；这是和“通晓未来”一样能够帮我们找出现下正在犯的错误的方法
  • 古人和我们是一样的人，不管他们产生了什么想法，那都是正常人产生的想法
  • diff观点：当代和不同时期的古代、当地和各地——做到这点最好的办法就是亲自去看

第四种方法：道貌岸然

  • 寻找那些一本正经的卫道士，看看他们到底在捍卫什么
  • 孩子眼里的世界是不真实的，是一个被灌输进他们头脑的假想世界
  • 大多数成年人故意让孩子对世界有一个错误的认识，即使是如同“圣诞老人”这样的事，到底是为了孩子，还是为了自己？
  • 比较一个历经很多事的人和一个稚嫩的孩子的思想，相减就能知道哪些是“不能说的话“了

第五种方法：禁忌是如何产生的

  • 某种禁忌、道德观念是怎么产生的？又是怎么被其他人接受的
  • 流行时尚是不一样的，一般来说，时尚来自某个有影响力的人物，来自某个偶然
  • 流行的道德观念不一样，并非偶然，而是刻意被创造，往往来自某个团体——这个团体不能弱小，否则无力推行；也不会很强大，否则就不会在乎
  • 道德禁忌的最大制造者，是那些在权力斗争里略占上风的一方
  • 大多数斗争，不管实际上争斗的是什么，最终都会以思想斗争的形式表现出来——因为思想斗争更容易争取支持者
    1. 第一批接受者总是带有很强的抱负心，有自觉的精英意识，想把自己和普通人区分开
    2. 第二批人会在流行趋势确立之后加入，驱使他们的是恐惧心，他们不是想要与众不同，而是害怕与众不同
  • 观察
    1. 哪个团体势力强大，却又精神高度紧张，喜欢压制什么样的思想观点
    2. 近期有没有什么社会斗争，失败的是哪一方，受他们牵连的是什么思想观点
  • 当然也不要忘记，有些禁忌不是社会斗争产生，而是植根于过去

为什么要去分辨和观察

  • 好奇的天性
  • 不喜欢犯错，不喜欢上当
  • 很好的脑力训练——你需要一个什么问题都能思考的大脑，尤其是那些似乎不应该思考的问题，你的大脑需要养成思考的习惯
  • 优秀的作品往往来自其他人忽视的想法，而最被忽视的想法往往就是那些被禁止的想法
  • 科学理论上，寻找传统观点无法自圆其说的地方，试着拆开里面寻找问题，这就是新理论诞生的开始，科学家就是喜欢自找麻烦，这也应该是任何学者的研究方式
    1. 在科学界你要磨练自己的智力，发现解决真正的问题；而不是像政治家通过搞人事关系和派系斗争立足
    2. 智力越高的人越愿意去思考惊世骇俗的思想观点，这不仅是因为聪明能想到，更因为是传统观念对他们的束缚很小，很容易摆脱
    3. 你能看到别人看不到或者不敢看的东西，你就有很大的优势

如何对待“不能说的话”

守口如瓶

  • 别说！至少也要挑选合适的场合再说，只打那些值得打的仗
  • 如果你的目的是争论，那么说出来和别人争论正中你下怀；但如果不是，变成别人眼里的对立面，对你真正想要干的事就成为了极大的干扰。和笨蛋争论，你也会变成笨蛋。
  • 自由思考比畅所欲言更加重要，在思想和言论之间应该画一条线，心里无所不想但是不一定要说出来
  • 守口如瓶，笑脸相迎——你要对每一个人微笑，但是不要说出真实的想法。“格斗俱乐部”的第一条规则，就是不要提到“格斗俱乐部”
  • 每个时代都有忌讳，触犯他们重则坐牢，轻则惹来麻烦，干扰生活。口无遮拦，你就没空做正事了。为了和别人论战，你不得不变成一个语言学家。
  • 能够一起谈论“异端邪说”并且不会因此气急败坏的人，是你最应该认识的朋友；和他们讨论一个观点，产生更多的观点——这是“讨论”本身的好处；这样不仅可以获得新观点，也能选择朋友。

笑脸相迎

  • 不赞同这个时代的任何一种歇斯底里，但是又不明确告诉别人到底不赞同哪一种歇斯底里。
  • 狂热分子会试图诱出你的真心话，但你可以不回答。如果他们不放手一定要你表达观点，那你可以“既不赞成也不反对“，当然，更好的选择是说”我还没想好“
  • 人们喜欢讨论的问题，实际上都是很复杂的，马上就说出你的想法其实对你没什么好处
  • 避免被贴标签：
    1. 逐步把辩论上升到一个抽象的层次
      • 避免对手抓住你的具体，声称你支持的其实不是某某，而是被反对的具体某某内容
      • 不要直接攻击标签本身，而要攻击“元标签”——对某个标签的抽象描述。
      • 如果人们开始讨论元标签，那么原来的标签反而不受注意了
    2. 使用隐喻
    3. 幽默
      • 狂热分子都有一个共同点：缺乏幽默感，无法平静的对待笑话

永远质疑

  • 没有人认为自己思想不开放，然而实际上，人们的脑子里有一根线，早就认准了什么是对的什么是错的
  • 再次重申：所谓的流行，本质上就是自己看不见自己的样子
  • 对于被流行抓住的人，流行就是应该做的事
  • 保持距离才能发现流行
    1. 时间产生距离：新的流行让旧的流行更容易被发现
    2. 远离人群产生距离：特别注意那些被压制的思想观点
  • 如果一个问题是错的，那么它不需要被贴上标签，被贴上标签各种压制，就是明确的信号，表明背后有问题
  • 远距离观察你自己
    1. 自己就是社会潮流的一部分，所以要永远保持质疑
    2. 孩子精疲力竭会大发脾气，因为他不知道为什么会这样；成年人则会认识到这是自己身体状况的问题，和外界无关——“我只是累了”
    3. 通过类似的机制，我们应该可以分离外界的流行和内心的世界。
