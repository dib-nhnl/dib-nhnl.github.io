黑客与画家（4）

发表于 2019-08-26
本文字数： 424 阅读时长 ≈ 1 分钟

良好的坏习惯

不服从管教

  1. 黑客的主要字面意思是“精通”
  2. 黑客不服从管教，但是有些人的这种态度是装出来的（年轻人模仿好显得自己更聪明，这种装出来的不服从加上故作姿态挑毛病的态度令人恼火，而且其实延缓了创新的进程）
  3. 黑客不喜欢版权，而事实上，在计算机工业的历史上，新技术往往是由外部人员开发的
    • ibm在开发下一代电脑，乔布斯他们却做出来了
    • 计算机工业的几大巨头在研发新的一版操作系统，unix又诞生在两个年轻人手里
    • 版权法禁止外部人员了解专有技术的内部细节，也就禁止了外部人员由此产生新构想
  4. 硅谷出现在美国，是因为不同于英法德日，美国人本性上不服从管教。

自负和自由

  1. 黑客总是很自负
  2. 公民自由是国家富强的原因而非结果
  3. 决策不应该总从权势最大的人手里产生
  4. 专制带来腐败，腐败带来贫穷，贫穷带来弱小
  5. 极权制度一旦形成就难以废除，限制自由，政府的力量会先增大后减小
  6. 极权主义，会导致一个错误观点占上风的世界

American-ness

对美国精神的自豪
